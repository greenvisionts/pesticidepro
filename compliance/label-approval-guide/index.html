Pesticide Label Approval in India — CIB Process and Requirements

Every pesticide label in India requires CIB approval before use. Complete guide to mandatory elements, approval process, common rejection reasons, and multilingual requirements.

Try CIB Compliance Software →

Mandatory Label Elements

Required: (1) Brand name at top, (2) Common and chemical name + percentage, (3) Formulation type, (4) Net contents, (5) Batch number and dates, (6) Registration number, (7) Hazard classification with red triangle/blue diamond, (8) First aid in bold, (9) Directions for use table (crop/pest/dose/water/timing), (10) Waiting period per crop, (11) Manufacturer name and address, (12) CIB/RC registration number prominently.

Approval Process Timeline

Submit to CIB-RC: product registration copy, label artwork (all language versions), reference to relevant Insecticides Rules. Technical committee review. Typical timelines: 3–6 months for straightforward labels, 6–12 months if revisions needed. New uses or scheduled/banned molecules require more scrutiny.

Multilingual Requirements

Labels must be in English or Hindi plus 2 regional languages of main selling states. Product sold in Maharashtra, AP, Karnataka needs: English/Hindi + Marathi + Telugu + Kannada. Minimum font sizes prescribed by CIB for each mandatory element.

Common Rejection Reasons

Most common reasons: (1) Missing first aid or antidote instructions, (2) Font size below CIB minimum, (3) Wrong hazard symbol for toxicity class, (4) Uses not covered in registration, (5) Claims exceeding registration data, (6) Missing re-entry interval (REI), (7) Waiting period not specified per crop.

Label Amendment Requirements

Any change to active ingredient, concentration, claims, or safety information requires fresh CIB approval. Cannot self-certify changes. Process same as original approval. Maintain archive of all approved label versions — required during inspection.
